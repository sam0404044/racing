改版說明 — 從「算速度」到「踩 vs 收」v0.6 → v0.9

→ 完整規則（GAME_RULES） → 機制細節（GAME_MECHANICS） → 賽道設計（TRACK_DESIGN） → 對手設計（OPPONENT_DESIGN） → 卡牌設計（CARD_DESIGN）

PART 1設計理念轉換

這次我把上一版的核心張力反過來：以前是「怎麼換算手牌價值把速度湊到能超車？」，現在是「我需要更多或是減少速度？」

團隊回饋的兩個核心點：

  • 「要有速度感」
  • 「不要太像數學遊戲」

舊版用比較經典的卡牌遊戲思路：限制資源、要玩家仔細衡量手裡的東西換算成速度才能取得優勢。但實際遊玩會產生兩個問題——手牌很卡，以及玩家會覺得速度取得不易。這跟我們想給玩家的「速度感」是相反的。

所以這版我轉了個彎：讓玩家獲取速度非常容易，反而要思考的是「我現在要多一點還是少一點速度」。能量機制被移除，速度取得變得很流暢，決策點轉移到「衡量速度的多寡」上。

三個系統互鎖的新核心

⚡
速度（易得難管）
將能量系統拿掉。打牌幾乎零代價、賽道加成、尾流，速度便宜累積。但太快 QTE會變太難、太慢又會無法超過前車。
🛞
輪胎（血量化）
取代舊版的能量，5 層胎 = HP。主動衝速（氮氣噴射）跟 QTE 失敗都會扣胎，扣到 0 就直接被淘汰。呼應賽車氛圍。
🃏
失誤牌（雙向作用）
起始牌庫拿掉所有失誤牌。QTE 失敗會收失誤牌污染牌庫頂；Perfect 反而會清掉一張失誤牌。獎勵把 QTE 玩好的玩家。

逐項改動清單

① 行動 vs 回合 — 速度感的來源

定義兩個層次：

  • 一個「行動」：打一張牌、或換一次道。每個行動都讓車子前進一個賽段。
  • 一個「回合」：發手牌開始，到玩家按超車 / Pass 結束。每回合通常會有好幾個行動。

因為每打一張牌就切下一個賽段，速度不斷地在賽道間累加，速度感馬上就有了——這是這版最關鍵的視覺與節奏改動。

② 彎道限速 + 彎道 QTE

進到「彎道」類賽段、玩家速度超過該段限速 → 觸發新的彎道 QTE（WASD 箭頭串，仿 Helldivers stratagem 風格）。

  • 內彎：速度加成大（×1.25）、但限速低（75）
  • 外彎：速度加成小或負（×0.9）、但限速高（105）

QTE 失敗 → 扣 1 胎 + 抽 1 張失誤牌。直線段不檢查限速，玩家可以放心衝速——所以彎道 QTE 是玩家自己選的，不是被強加的。

③ QTE 越快越難

QTE 的難度跟玩家當下速度綁定——速度越快、節奏越緊、箭頭越多、計時越短。這讓「該加速還該收一點」變成貫穿整局的張力 。

④ 失誤牌的雙向作用

  • 起始牌庫：8 張全是有效牌（turbo ×2 + tailwind ×3 + drag ×2 + laneRhythm ×1）、無失誤牌污染
  • QTE 失敗 → 失誤牌進牌庫頂、下回合一定抽到
  • QTE 100% Perfect → 從牌庫/棄牌堆移除 1 張失誤牌

這套機制讓玩家有理由主動拚 Perfect（清牌組），而不只是「不要 Miss 太多」。

⑤ 輪胎 = 血量（取代能量）

輪胎不再是「打某些牌要付的資源」這種無聊的能量替身——它就是 HP。

  • 5 層胎、扣完 = 車輛打滑失控、比賽淘汰
  • 扣胎來源：氮氣噴射、換道節奏（主動）、超車 QTE 失敗、彎道 QTE 失敗
  • 跟「紅牌捨身攻擊」設計直接對應——犧牲血量換爆發加速，呼應賽車氛圍

⑥ 多賽道 + 同道尾流

每個賽段有幾條賽道，玩家打牌時打在哪條道就直接吃那條道的加成（加成設計大多是 +/- 號，不是複雜的乘除）。對手免疫所有賽道加成——加成只幫玩家、不幫對手；但賽道災害（c6 油污、c7 坑洞）對雙方都生效。

  • 換道代價降低：不再扣能量/輪胎，改為扣 speed（公式 10 + 5 × (跨道數 - 1)）。打牌的代價低、換道的代價也低 → 每回合的行動數變多。
  • 尾流：每回合第一次跟對手同道時 +30 速。賽車中的標誌性行為，多賽道機制應該讓它鮮明地浮現出來。

⑦ 超車 = 必須切道、必須 QTE

玩家必須跟對手在不同道才能超車——好比對手一定會擋你，所以每次超車都會有精彩的車手技術 QTE。

⑧ 對手專注值（不只 Boss 有）

我覺得「磨對手」的張力不該只在 Boss 戰才出現，所以一般對手也有 1~2 格專注值。

  • 1 專注 = 要 2 次成功超車 QTE 才能甩開
  • 2 專注 = 要 3 次

呈現上會有那種「我在左道吃尾速、切到右道要超車、進到 QTE 他來阻攔、但他的專注就渙散了、下一次我又切左道超車就成功」的感覺。

⑨ 對手雙行動 + 意圖提示

每個對手有弱招 + 強招兩套行為：

  • 弱招：CD 短、通常跟移動有關
  • 強招：CD 長、跟角色性格綁定（破風者狂加速、清道夫鎖住道、禿鷹遠離又貼上）

UI 會在名條面板顯示對手「再幾個行動後出招」+ 移動意圖 icon，但不會告訴你細節：

  • ⛔ 阻擋你：對手會切到你動作前的道
  • 💨 遠離你：對手會避開你的道（破壞尾流）
  • ❓ 不確定：看當下道路最快、或隨機切道
  • ❗ 賽道干擾：對手原本想 ⛔ / 💨、但賽段有坑洞或油污、AI 會繞開、結果偏離原意圖

關鍵設計：對手用「玩家動作前所在道」當基準預測——所以玩家可以用換道閃過對手的 ⛔ 阻擋。（對手免疫賽道加成、不會被「釣進逆風道」，但對手 AI 會自動避坑、避免進入坑洞道。）

⑩ Pass 解決舊問題

之前團隊提過 Pass 按鈕的存在感太弱。新框架下 Pass 直接等於防守候車——按 Pass 直接進防守 QTE、失敗掉名次。這個問題就解掉了。

⑪ 留牌補手（取代清手牌）

回合結束不清手牌，從牌庫補到 5 張。這跟「鼓勵玩家策略性規劃後手」綁定——濾牌就變得有意義（看到下個賽段有特殊加成、可以把大牌留下來等彎道）。

⑫ 對手免疫賽道加成

對手不再吃任何賽道的 add / mult / speedLimit——加成只屬於玩家。對手速度的唯一來源是它自己的動作（boost / 切道 + boostAfter / stay+absBonus）。但賽道災害（c6 油污、c7 坑洞）仍對雙方有效——賽道對對手變成「中立風險」、不會幫忙、但會懲罰。

例外：清道夫光環免疫所有災害——當前對手是清道夫時、它踩坑不降速、踩油污不滑開。

這個設計解掉「對手在彎道反而加速」、「賽道機制反幫對手」的舊問題；同時讓賽道戰術完全屬於玩家、決策清晰。

⑬ 卡池全面重設計

指令牌從 3 張擴到 9 張，數值光譜更密、每張牌有清楚的「為什麼選它」：

  • 純數值：turbo +30、tailwind +20、drag +15
  • 換道：laneRhythm +15 + 可換道 + 消耗 1 胎
  • 爆發 + 代價：nitro +60 / -1 胎、reignite +25 / 下回合手牌 +1
  • 條件 / 賭注：drift（僅彎道、必觸 QTE）、chill（QTE 容錯 +50%）、smoothOp（連發加倍）

車隊牌新增 7 張、用雙維度描述（trigger = equip / play、persistence = permanent / oneShot / untilRankUp / thisRound）。打出車隊牌不算行動——不切賽段、不觸發對手、像「無線電指令 / 免費裝備」。

⑭ 10 回合終點線

沙盒對戰有回合上限 10。跑到第 11 回合就強制結束、依當前名次結算。這把原本「無限延伸到三人都被超過」的開放結尾改成有壓力感的競賽：每回合都重要、第 8/9/10 回合會變得緊張。HUD 上會顯示「回合 X / 10」，最後 2 回合變紅警示。

實際玩起來，玩家的注意力會這樣分配：

  • 「下一個賽段是內彎還是順風？」 → 規劃手牌節奏
  • 「我這回合是要切道吃對手尾流，純加速最大化、還是遠離準備超車？」 → 不同道戰術
  • 「對手強招還剩幾個行動？他會阻擋還是遠離？」 → 預判博弈
  • 「我這階段該再衝一張、還是直接超？」 → QTE 風險評估
  • 「我胎只剩 2 層了，氮氣噴射還划算嗎？」 → HP 管理
玩家的關注點變多，而且沒有一個是「算數字能不能超過別人」——加上輪胎這個淘汰風險，玩家還會被迫思考：超太快 → QTE 失敗 → 破胎，是不是該收一點？

當然你還是需要速度超過對手才能挑戰超車——但這已經不是計算門檻，而是在彎道、尾流、對手意圖、輪胎血量之間做動態取捨。這就是這套機制成熟的地方。

PART 2給企劃團隊的試玩注意事項

三句話總結

  1. 速度便宜，要規劃的是什麼時候收、什麼時候加?——超速會 QTE ➔ 越快QTE越難 ➔ QTE失敗會破胎。
  2. 換道是博弈、不是負擔——尾流、超車、避開對手都靠它。
  3. 看下一賽段預告、看對手意圖 icon——這是兩個最重要的 UI 資訊源。

玩前一定要知道的 8 件事

1. 一個行動 = 打一張牌或換道

不是「我打到滿意才結束」——每打一張指令牌或換道，車子就前進一個賽段。關注賽段預告，你才知道下一段有什麼道路、合理安牌手上的牌。
例外：車隊牌打出不算行動、不切賽段、不觸發對手——它是「免費裝備」。

2. 換道很便宜、是核心策略動作

換道代價只有 -10 speed（跨 1 道）。不要只想直線加速——切道吃尾流、切道找順風、切道準備超車，都是這版的常態。

3. 同道才有尾流（每回合一次）

跟對手同道時 +30 速、但對手會躲。如果對手有 ⛔ 阻擋意圖、你又貼上去 → 對手會切走你就吃不到尾流（還會被嘲諷）。如果對手有 💨 遠離意圖、你可以反向預判。

4. 超車必須不同道

同道無法按超車鈕。切到不同道、speed > 對手 → 才能按超車鈕進 QTE。

5. 對手要磨、不是一次超

對手有 1–2 格專注。超車 QTE 成功 → 扣 1 專注；專注 = 0 後的下次成功才真的名次 +1。一場對戰會有 2–3 次超車 QTE。

6. 速度太快會自找麻煩

  • 進彎道賽段、speed > 限速 → 觸發彎道 QTE（WASD 串）。失敗扣胎 + 抽失誤牌。
  • QTE 整體會隨速度變難——速度越快、節奏越緊。

7. 起始牌庫沒有失誤牌

失誤牌只從 QTE 失敗收。所以盡量把超車 QTE 打到 Perfect——還能反向清掉之前累積的失誤牌。

8. 看對手名條的意圖 icon

  • ⛔ = 對手下次會切到你目前的道（阻擋）
  • 💨 = 對手下次會避開你目前的道（破壞尾流）
  • ❓ = 不確定（看當下狀況決定）
  • ❗ = 賽道干擾（對手原本意圖被坑洞 / 油污打亂）
  • ⚡ = 強招（會額外加速、或有特殊效果）

這個 icon 顯示在名條面板 + 切道紅 banner，是你最重要的預判資訊。

常見誤區

誤區 1：「我只想無腦打加速牌、衝過對手」
→ 速度會被干擾不夠、或是被前車阻擋。試著切道吃尾流、或切到順風道再衝。
誤區 2：「我把所有手牌打完才超車」
→ 一直打牌就一直切賽段、速度疊得很高但是 QTE 滑鐵盧，得不償失。看下一段預告，該收就收、該按超車就按。
誤區 3：「超車 QTE 過關就算贏」
→ 60% 過關只是磨對手 1 格專注。要超車成功、要先磨光對手的專注。
誤區 4：「胎還很多、氮氣噴射隨便用」
→ 氮氣噴射 -1 胎 + QTE 失敗還會再 -1。一場下來 5 層胎扣很快。
誤區 5：「對手永遠都會擋我」
→ 對手不一定阻擋。看意圖 icon——⛔ 才是阻擋、💨 是遠離。
對手免疫所有賽道加成、所以「釣對手到逆風道」沒效。但你仍可以利用閃避（換道）來破壞對手的 ⛔ 阻擋。

PART 3Core Gameplay Loop

玩家在每一個回合內、會反覆執行多個行動，最後決定按超車或 Pass 結束回合。

視覺化循環圖

文字版本（給文件閱讀）

  1. ① 回合開始
    • speed 重置為起始值（預設 10）
    • 手牌補滿 5 張（沒打完的留著）
    • 看下一賽段預告 + 對手名條的意圖 icon ← 規劃資訊源
  2. ② 思考：踩 vs 收
    • 下一段是順風 / 彎道 / 油污 / 紅綠燈？該打哪張牌？
    • 要切道吃尾流？還是切道準備超車？
    • 手上的大牌要留到下一段嗎？
  3. ③ 執行一個行動 → 走完五步驟流程
    1. 玩家行動：打牌 → 加 speed；換道 → 扣 speed + 棄 1 張
    2. 對手行動：依弱招/強招冷卻、可能切道或加速
    3. 檢查尾流：同道 → +30 speed（每回合一次）；對手躲開 → 觸發嘲諷
    4. 賽道結算：套用所在道加成；玩家超限速 → 觸發彎道 QTE
    5. 切下個賽段（影響下個行動）
  4. ④ 評估：繼續再一個行動，還是結束回合？
    • 還有手牌、還想累積 → 回到 ②
    • 速度夠、跟對手不同道 → 走 ④A 按超車
    • 不想衝、想下回合再來 → 走 ④B 按 Pass
  5. ④A 超車 → 超車 QTE → 成功扣對手 1 專注；專注 = 0 後的下次成功才名次 +1。Perfect 還能清失誤牌、低於 60% 失敗扣胎收失誤牌。
  6. ④B Pass → 直接進防守 QTE。失敗掉 1 名（最後一名免疫）。
  7. 結算名次後進入下一回合 → 回到 ①

這版的目標是讓玩家在回合中的，每一個行動都要規劃——不是算價值，而是在這條彎道、這個對手、這層輪胎之間做動態取捨。試玩時請特別觀察：

  • 你的注意力大部分時間是在「算數字」還是在「做選擇」？
  • 你是不是有種「下一段我要保留這張牌」的策略感？
  • 對手的意圖 icon 你看了幾次？你有沒有想去釣他到差賽道？
  • 輪胎扣到剩 2 層時、你的決策有沒有變保守？

這些問題的答案會直接告訴我們，「踩 vs 收」這個張力到底有沒有真的浮現。
