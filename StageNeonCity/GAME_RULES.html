最後車手 / Final Driver 遊戲規則v0.6 — 霓虹多線街區

這一版的核心設計從「被限制的速度取得、計算超車門檻」轉向「加速容易、反而要規劃降速」。玩家不再卡在「速度不夠 → 超不過 → 下回合歸零」的挫敗迴圈，而是全程在「踩 vs 收」之間做決策。

→ 機制細節（GAME_MECHANICS） → 賽道設計（TRACK_DESIGN） → 對手設計（OPPONENT_DESIGN） → 卡牌設計（CARD_DESIGN）

想看精確數值、公式、卡牌池請看 MECHANICS；對手與 AI 細節見 OPPONENT_DESIGN、卡牌細節見 CARD_DESIGN、新賽段提案見 TRACK_DESIGN。

關卡架構：此版本暫停教學關卡 1–4 的迭代，新增單一沙盒關卡「霓虹多線街區」作為新核心機制的測試場。每張牌都會觸發事件，完全沒有「無事發生」的回合。

遊戲目標

賽道共有 4 位車手，玩家從最後一名（第 4 名）開始。透過打牌指揮車手累積速度、操作三種 QTE，逐步超越對手取得第 1 名。

行動與回合

  • 一個行動：玩家打一張牌或換道，就算一個行動。每個行動會走完五步驟流程（含對手回合、賽道結算、切段）。
  • 一回合：從發手牌開始，直到玩家按超車 / Pass 為止，叫一回合。每回合通常包含多個行動。

這兩個層次很重要：切賽段、賽道結算是每行動一次；速度重置、尾流可用是每回合一次。

核心循環

  1. 每回合速度重置為起始值（依車隊牌 tireWarm 等加成、預設 10）、發手牌。
  2. 每個玩家動作（拖牌、按超車、按 PASS）執行五步驟流程：玩家行動 → 對手行動 → 檢查尾流 → 賽道結算 → 賽段長度計數 -1（歸 0 才切下個賽段、否則停留當前段）。
  3. 跟對手同道時 +30 速（每回合一次、超車/PASS 進新回合重置）。
  4. 想超車：不同道 + 速度 > 對手 → 按「超車」鈕 → 超車 QTE → 成功 = 扣對手 1 專注，專注 = 0 時下次成功才名次 +1。
  5. 進到「彎道」類賽段且玩家速度 > 限速 → 觸發彎道 QTE。
  6. 不想衝就按 Pass → 直接進防守 QTE。
  7. 結算名次後進入下一回合（speed/手牌重置、賽段保留接續）。

核心規則：速度必須嚴格大於對手才能挑戰超車——平手不算。同道不能超車——要先換到不同道、再追上速度。

名次規則

  • 初始名次：4 / 4（共 4 位車手，從最後一名開始）。
  • 成功超車 + 對手專注 = 0：名次前進 1 名（詳見下節）。
  • 防守 QTE 失敗：名次後退 1 名（最低為第 4 名）。
  • 到達第 1 名 = 遊戲勝利。

第 4 名後方沒有追車，Pass 後不會觸發防守，也不會繼續下掉。

對手專注值 NEW

每個對手都有「專注值」（類似耐性 / HP）。成功的超車 QTE 並不等於名次前進——你必須先磨光對手的專注，才能真正甩開他。

  • 對手的專注值通常是 1 或 2 點（依對手強度設定）。
  • 每次成功超車 QTE（≥ 60%）→ 扣 1 專注（除非有特殊效果）。
  • 專注 > 0 時：超車 QTE 成功只是「壓制」，名次不變。
  • 專注 = 0 後：下一次成功超車 QTE = 名次前進 1 名。
  • 超車失敗（< 60%）：扣胎、收失誤牌，但不影響專注。

實際遊玩：1 專注的對手要 2 次成功超車 QTE 才會被甩開；2 專注的對手要 3 次。這讓對手變成「卡位戰」對象，玩家在這幾回合內會持續同道、換道、超車。

輪胎系統 NEW

取代舊版的「動力」，輪胎是這版的核心資源。

  • 起始 5 層胎，扣到 0 = 輸。
  • 每張卡牌的代價已內建（速度、失誤、QTE 風險），打牌不直接扣胎。
  • 扣胎來源：超加速（主動）、超車 QTE 失敗、彎道 QTE 失敗（被動）。
  • 防守 QTE 失敗不扣胎（已有掉名次的懲罰）。

補胎機制（Pit Stop）規劃中，先不實作。精確扣胎數值請見 機制細節。

速度系統

  • 每回合速度重置為起始值（預設 10、車隊牌可加成）。
  • 普通牌 = 加速度（無代價）：加速 +20、超加速 +40、渦輪 +30。
  • 超加速 = 強力加速但 -1 胎。
  • 速度結算分兩步：步驟 1 行動效果立即生效（cardValue、laneCost）；步驟 4 賽道結算才套順風加成 / 彎道加乘 / 速限。
  • 賽道加成與災害的雙重規則（v0.8 起）：
    • 加成（順風 / 逆風 / 彎道 mult / speedLimit）：只影響玩家、對手免疫。玩家用 (speed + add) × mult 結算、超速限觸發彎道 QTE。
    • 災害（c6 油污 50% 滑開 / c7 坑洞速度減半）：雙方都會中招、不偏袒對手。
    設計意圖：賽道對對手是「中立風險」——不會幫忙、但也會懲罰。詳見 對手設計。

同道尾流 NEW

玩家跟對手同道時 +30 速，作為「卡位」的獎勵。

  • 觸發時機：步驟 3 檢查時（對手行動完成後）、玩家當前道 == 對手當前道 → +30 速。
  • 一回合只觸發一次——超車/PASS 進新回合才重置。
  • 對手能躲開尾流：玩家動作後若同道、對手在步驟 2 可能切走，玩家就吃不到 +30。
  • 玩家想吃尾流但對手躲開時 → 出現對手嘲諷文字（依對手個性不同）。

例：玩家換道貼上對手（步驟 1）→ 對手立刻切走（步驟 2）→ 步驟 3 不同道、沒尾流 → 顯示嘲諷「對手讀出你的尾流意圖」。

換道規則

  • 棄一張手牌 + 依跨道數扣速：跨 1 道 -10，跨 2 道 -15，跨 3 道 -20 （公式 10 + 5 × (跨道數 - 1)）。
  • laneCost 在步驟 1 立即扣 playerSpeed。
  • 速度 0 時不能換道。
  • 換道本身不消耗任何 QTE，但仍會走完五步驟流程（含對手回合、賽道結算、切段）。

賽段切換 NEW

每打一張牌（或換道），自動切換到下一個賽段。UI 會顯示「當前賽段」與「下一段預告」，方便提前規劃。

賽段的核心概念：

  • 賽段 = 結構 + 多條賽道。結構決定有幾條道、能不能換；賽道則各自有加成（速度 +N / -N / ×N）與事件（如彎道限速）。
  • 車子在哪條賽道，就吃那條賽道的加成——加成是車子的持續狀態，玩家跟對手都吃。
  • 顯示速度 = 累積速度 + 當前賽道加成。所有「限速」、「對手速度」、「超車條件」的比較，都用加成後的顯示速度。

選道本身就是博弈：同樣是 60 vs 60，若玩家在順風道、對手在逆風道 → 玩家 70 vs 對手 50 → 可以超車；反之則被甩開。完整的賽道類型、賽段定義與數值請見 機制細節。

超車規則 NEW

對手一定會擋你，所以所有超車都要 QTE。

  • 同道不能超車——必須先換道。
  • 觸發方式：玩家手動按「超車」按鈕。
  • 按鈕啟用條件：不同道 + 速度 > 對手。
  • 不自動觸發——玩家保留「現在拚還是再積一張？」的決策權。

三種 QTE

超車 QTE（節奏點擊）

  • 機制：5–8 個節奏圓圈（依速度檔位），玩家在圈收縮接近拍點時點擊。
  • 判定：Perfect（2 分）/ Good（1 分）/ Miss（0 分）。
  • 60% 為過關門檻；< 60% 為超車失敗。

結果分階

分數失誤牌結果扣胎
100%-1 張（移除）完美超車否
70–99%0 張順利超車否
60–69%1 張勉強過關否
50–59%1 張輕度失敗是
< 50%2 張嚴重失敗是

失誤牌進牌庫頂，下回合必抽到。完美超車不只零代價，還能從牌庫或棄牌堆中移除一張失誤牌（若無失誤牌可移除則無效果），玩家會主動拚 100% 清牌組。

彎道 QTE（WASD 箭頭串） NEW

  • 機制：仿 Helldivers stratagem 風格——螢幕顯示一串箭頭（↑ ↓ ← →，對應 W S A D），玩家依序按出。
  • 難度隨速度提升：速度越快，箭頭越多、計時越短。
  • 觸發條件：只在「彎道」類賽段檢查限速，速度 > 該段限速才觸發。
  • 失敗 = -1 胎 + 1 張失誤牌（進牌庫頂）。

直線段不檢查限速——玩家可以放心衝速，QTE 變成玩家自己選的，不是被強加的。

防守 QTE（追游標）

  • 機制：玩家移動滑鼠，讓黃色游標追住綠色安全區。
  • 游標在安全區內 → 進度增加；在完美區（綠區中央）→ 增加更快；在紅區 → 進度下降。
  • 撐到時間結束或進度推滿 = 防守成功。
  • 觸發：玩家按 Pass 後直接觸發（不再判斷後車速度）。
  • 失敗 = 掉名次（不扣胎）。

Pass 規則

  • Pass 後直接進防守 QTE。
  • 拿掉舊版「後車速度 > 玩家速度才防守」的判斷——Pass 一定觸發防守 QTE。
  • 失敗掉名次。
  • 後車追擊的驚嘆號 UI 保留，作為壓力氛圍（純視覺）。

卡牌規則

卡牌類別

類別定位消失時機
指令牌 CMD立即指揮車子的行動（加速、過彎等）打出後進棄牌堆
車隊牌 TEAM影響狀態或駕駛特性，留場持續生效依牌面所述
失誤牌無效果，污染牌庫打出後進棄牌堆

手牌與牌庫運作

  • 每回合手牌上限 5 張。
  • 回合結束時不清空手牌，只從牌庫補到 5 張——沒打完的牌可以留到下回合用。
  • 採用真實牌堆機制：打出的牌進棄牌堆，抽完才把棄牌堆洗回牌堆。
  • 失誤牌進牌庫頂，下回合必抽到。每抽到一張失誤牌就少一張可用牌。

完整起始牌庫、所有指令牌與車隊牌的列表請見 機制細節。

霓虹多線街區（沙盒測試版）

單一關卡，無教學引導，整合上述所有機制。設計重點：

  • 多賽道：強制玩家在「同道吃尾流」與「不同道準備超車」之間切換。
  • 賽段循環：直線 / 彎道 / 其他類型輪替（具體節奏迭代中）。
  • 對手會反應：依照 cooldown 動態切道，名條面板顯示移動意圖 icon：⛔（阻擋你）／💨（遠離你）／❓（策略不確定）／❗（賽道干擾結果偏離），玩家可預判並換道閃過。
  • 對手會嘲諷：玩家想吃尾流但對手切走時，依對手個性顯示不同嘲諷文字（破風者技術派 / 清道夫陰沈 / 禿鷹得意）。

注意：此版本以核心機制驗證為目標。視覺、UI、平衡數值都還在迭代中，目標是禮拜五前收斂出大方向。
